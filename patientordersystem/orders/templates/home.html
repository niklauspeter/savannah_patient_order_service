Welcome to the Patient Order Service

This interface allows you to test our APIs via your browser (through django's inbuilt DRF interface).

However, as you are not authenticated, you won't be able to access the permission protected apis (add customers & add orders)

  • Start by signup/login to the service via AuthO's platform (if you're not signed in already)

  • Access tokens received from the AuthO's OpenId connect (OIDC) authentication will then be stored in your browser session, allowing you to interact with the API endpoints listed below.

  • On clicking the various endpoints(or typing them out on your browser search bar), you'll be redirected to django's inbuilt API testing interface where you can send and receive requests (Debug is set to true purely for testing)

Available Routes

  • Add Customer: /api/customers/add/
  • List Customers: /api/customers/list/
  • Add Order: /api/orders/add/
  • List Orders: /api/orders/list/
